# CalculadoraApp

Aplicação iOS desenvolvida em Swift para realizar a soma entre dois números.

## Funcionalidades
- Entrada de dois números
- Cálculo de soma
- Exibição do resultado
- Validação para aceitar apenas números
- Interface simples e organizada

## Tecnologias
- Swift
- UIKit

## Objetivo
Projeto desenvolvido para prática de lógica de programação e integração entre interface e código no desenvolvimento iOS.
